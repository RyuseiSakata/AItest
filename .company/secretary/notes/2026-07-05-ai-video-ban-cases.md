# AI動画で「消された/収益停止された」実例集 — 禁止パターン集

出典: deep-researchワークフロー(2026-07-05実施)。23件の裏取り済み主張ベース。
一次情報=YouTube公式ヘルプ/公式スレッド、二次=Deadline・IBTimes・Hollywood Reporter・Forbes・講談社FRIDAY等。

## 結論(まず要点)
- **「AI使用」そのものは違反ではない**。消されているのは「AI丸投げの量産・テンプレ・欺瞞」の3類型。
- 措置は**3段階**でエスカレートする: ①リーチ抑制(可視性低下) → ②収益化停止 → ③チャンネルBAN(削除)。
- 最も多いのは**収益化停止**。BAN(完全削除)は「欺瞞・なりすまし・スパム」が絡んだ悪質例に集中。
- YouTubeの規制根拠は2025-07-15改称の「**inauthentic content(非真正コンテンツ)**」ポリシー。

---

## A. 海外の実名BAN事例(最も悪質 → 完全削除)

### 1. Screen Culture / KH Studio(偽AI映画予告編)
- 規模: 2チャンネル合計 **登録者200万人超・再生10億回超**。
- 内容: 公式スタジオ映像 + AI生成画像を切り貼りした偽の映画予告編。
  『ファンタスティック・フォー』予告を**23バージョン**量産し、一部は公式予告より検索上位に。
- 措置の経緯(エスカレーションの典型例):
  1. Deadlineの調査報道後、YouTubeが**広告停止(収益化停止)**
  2. タイトルに "fan trailer / parody / concept trailer" と付けたら**収益化復活**
  3. その注記をまた外した(=再び視聴者を欺いた) → **2025年12月チャンネル完全削除(termination)**
- 違反ポリシー: スパム + 誤解を招くメタデータ(misleading metadata)。
- 教訓: **「公式と誤認させる」「開示を外す」が最悪手**。開示ラベルの有無で運命が分かれた。

### 2. 2026年 大規模摘発(16チャンネル一斉)
- 規模: **合計登録者3,500万人・生涯再生47億回**、推定年間広告収益 約1,000万ドルが消滅。
- 措置内訳: **11チャンネル完全削除 + 5チャンネルは動画全消し(チャンネルは残す)**。
- 実名例:
  - CuentosFacinantes(登録者590万・ドラゴンボール系AI動画)
  - Imperio de Jesus(登録者580万・信仰系インタラクティブクイズ)
  - Héroes de Fantasía / Adhamali-0 / Roupa25
- 作り方の共通項: 合成音声 + 自動生成台本 + ストック/AI画像、人間の編集判断ほぼゼロの短尺テンプレ動画。
- 違反ポリシー: スパム・欺瞞的行為・非真正コンテンツ生産。
- ※実名リスト(CuentosFacinantes等)は検証2-1で確度やや低め。措置の全体像は確度高。

### 3. Noah Morris(faceless複数チャンネル運営者)
- 2025年初頭、YouTubeがfacelessチャンネルを停止し**月25万ドルの収益が消滅**。
- ただし停止のトリガーはAIポリシーでなく**著作権紛争** → 「AI規制とは別軸のリスク」の実例。

## B. 日本国内 — 2026年1月「停止祭り」
- 2026年1月、日本のYouTubeで**大量チャンネルの一斉収益化停止**が発生(通称「停止祭り」)。
- 措置は主に**収益化停止**(BAN・動画削除ではない)。
- 直撃ジャンル: 音声合成ソフト(ボイスロイド)を使った**「ゆっくり解説」など、顔出しなし・テンプレ構成**の動画。
- 背景: 規制対象の呼称が「Repetitive(繰り返し)」→「Mass-produced(量産型)」に変化。
  **同じ構成というだけで「機械的な大量生産」と判定されうる**ようになった。
- 復活方法(FRIDAY報道):
  - 公式ルート: 量産判定された動画を非公開化 / クリエイター本人の肉声を使う / 顔出し動画を追加
  - 裏技: X(旧Twitter)の @TeamYouTube に直接抗議し「人間運営」とアピール → 実際に復活例あり
- ※具体的なチャンネル名・登録者数は記事に記載なし。

## C. TikTok / Instagram

### TikTok(即ストライク方式へ転換)
- 2025年下半期に**合成メディア動画 51,618本を削除**(前年同期比 **+340%**)。
- **AI関連違反で 8,600アカウントを永久BAN**。
- 罰則は5段階エスカレーション: ①削除+ストライク → ②7日投稿制限 → ③30日制限 → ④永久収益化停止 → ⑤アカウント削除。
- 2025年11月、ユーザーがAIコンテンツの表示量を絞れる機能 + 不可視ウォーターマーク導入。
  → **ラベルを守ってもフィードで抑制されうる**(収益化以前の「可視性リスク」)。
- キモは「**未ラベルの合成メディア = 即削除対象**」。

### Instagram / Meta
- 主な措置はBANより「**AIラベル付与 + リーチ制限**」。
- C2PAメタデータで軽微な編集でも自動ラベル化。AIラベル付き投稿はエンゲージメントが下がるとの指摘(数値は業者ブログ由来で要割引)。
- Metaは2026年にInstagram/Facebook横断でAI開示を統一する方向。

## D. 禁止パターン集(これをやると消える)
1. **AI音声 + 静止画/スライドショーのテンプレ量産**(動画間の差異が表面的) ← 日本の停止祭りの中心
2. **公式素材の流用・切り貼りで本物と誤認させる**(偽予告編型) ← 最悪、完全BAN
3. **開示ラベルを外す/欺く**(一度付けた注記を消すのは特に危険)
4. **未ラベルの合成メディア**(TikTokは即ストライク)
5. **人間の編集判断がゼロの「AI丸投げ」**(自動台本+合成音声+自動画像)
6. 著作権紛争(AI規制とは別軸だが停止・削除の主要因。Noah Morris例)

## E. セーフ側の境界(これは維持できる)
- YouTube公式明言: **AIツールの使用自体はOK。AIを使うチャンネルも収益化資格を維持できる**。
  条件 = 収益化ポリシー遵守 + リアルに見える合成/改変コンテンツの**開示**。
- 生き残る鍵 = **「クリエイター本人のオリジナルな洞察・視点」を加える**こと(テンプレの逆)。
- 具体策: 肉声ナレーション/顔出し要素、動画ごとの構成差、独自の切り口・脚本、正直な開示ラベル。
- TikTok: AIキャプション・スクリプト補助・テキストオーバーレイはラベル不要(セーフ)。ラベル遵守なら案件収益は可能。

## F. 副業ジャンル選定への含意(→ vertical-video-side-businessに反映)
- 前回候補で最も危険なのは「テンプレ雑学(AI音声+ストック素材)」= 禁止パターン#1そのもの。
- **A2(テック/AI解説)**: 本人の視点・肉声・独自分析を乗せれば「オリジナルな洞察」条件を満たしやすい=生存性高。
- **D1(独自企画AIエンタメ)**: YouTube公式が「独自性あるAI活用は歓迎」。ただし欺瞞・なりすまし要素を絶対に入れない設計が前提。
- **B2(考察型ホラー)**: 脚本のオリジナリティが本体なので量産判定は受けにくい。ただし著作権(既存IP流用)に注意。
- 共通鉄則: ①肉声 or 独自視点を必ず入れる ②AI開示は正直に ③テンプレ反復を避け動画ごとに差を作る ④他者IP・公式素材の無断流用をしない。
