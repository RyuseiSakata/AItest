## これは何か
AI生成の縦型ショート動画で収益を作る副業プロジェクト。
仮想組織(.company/)で意思決定・調査・制作を管理し、ローカルPC(RTX 3060 Ti)+クラウドAIで動画を量産する。

## リポジトリ構成
```
├── .company/            仮想組織(秘書室・リサーチ部・制作部)— 意思決定ログと知識の本体
│   ├── secretary/       意思決定ログ・TODO・壁打ちメモ
│   ├── research/        調査レポート(裏取り済み)← まずここを読む
│   └── production/      チャンネル企画・台本・プロンプト・パイプライン
├── docs/                セットアップ手順・プロンプト知見(このフォルダ)
├── pipeline/            動画組み立てスクリプト(Python)と生成ジョブテンプレ
└── (root)               既存のRemotionプロジェクト(AIニュース動画の過去実験)
```
